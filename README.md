# ALEXANDRIA DIGITAL LIBRARY

A visually rich **digital library landing experience** built using **HTML, CSS, and vanilla JavaScript**, designed to showcase modern UI design, smooth interactions, and client-side data handling.

---

## Project Overview

**ALEXANDRIA DIGITAL LIBRARY** is a fully responsive, front-end digital library website built with HTML, CSS, and vanilla JavaScript. It features an elegant editorial-style design, advanced client-side search and filtering, interactive category navigation, recommendation sliders, and form validation — all implemented **without external JavaScript frameworks**.

The project focuses on **clean UI/UX**, **modern CSS design systems**, and **practical JavaScript logic**, including search algorithms, sliders, mobile navigation, and form validation.

> **Note:**  
> This is a personal portfolio project created for learning and demonstration purposes.

---

## Features
> **Note:**  
> Some features are currently being made.


- Elegant hero section with animated background and visual effects
- Fully responsive design (desktop, tablet, mobile)
- Sticky header with scroll-based styling
- Mobile navigation menu with outside-click handling
- Advanced client-side book search system
- Multi-parameter filtering (title, author, genre, year)
- Debounced live search for performance optimization
- Interactive category cards that trigger filtered searches
- Horizontally scrollable recommendations slider
- Slider navigation with state-aware controls
- Membership form with client-side validation
- Animated UI transitions and micro-interactions
- Accessible focus states and keyboard support
  
---

## Tech Stack

- **HTML5** — Semantic layout and structured sections  
- **CSS3** — Design tokens, gradients, animations, responsive layouts  
- **JavaScript (Vanilla)** — Search logic, filtering, sliders, navigation, form validation

No external frameworks or libraries are used (except icon fonts and web fonts).
